## Decisions and deviations

- I kept the product-gate Web build assertion inside the Desktop packaged smoke
  because `pnpm build:desktop` already owns the renderer build output. Adding a
  second production Web build in a parallel Web e2e file could race the same
  `apps/desktop-electron/dist/renderer` directory.
- I did not add release upload, signing, store metadata or telemetry.
