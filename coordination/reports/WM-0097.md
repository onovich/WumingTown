Web tier verdict: `demo-only`.

- Web is not cancelled. It remains a gated target in the Roadmap.
- Same-spec is not proven because the current browser path is a product-gate
  shell plus Worker projection path, not a measured 30 TPS / 20k-entity browser
  authority runtime.

  authority-runtime evidence.
- Demo-only is supported by Chrome/Edge shell launch, interaction, Web
  save/export/import, diagnostics, input and SharedArrayBuffer-unavailable
  fallback evidence.

- Ready means controlled sharing of the unsigned unpacked local directory build
  for launch, rendering, input, sandbox/preload boundary, product-gate surface
  and diagnostic-blocker testing.
- It does not mean installer, signing, updater, Steam/store, public release,
  crash upload, telemetry, account or paid-service readiness.
