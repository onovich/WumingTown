## Decisions and deviations

- ADR-0010 remains followed: M5 content is data, mutable authority remains in
  owner stores, and Worker/UI consumers remain read-only.
- `coordination/reports/WM-0083-plan.md` was added to the task allowed paths so
  the required execution plan is reviewable.
- The benchmark CLI default artifact root was advanced from WM-0070 to WM-0083
  so the required no-argument `pnpm bench` command writes the current closeout
  artifact instead of mutating the reviewed M4 artifact.
- `packages/benchmarks/src/m5-alpha-content-long-run-benchmark.ts` is 480 lines
  because it keeps the report contract, invariant contract, long-run sampling,
  save/replay evidence, Worker parity evidence and deterministic helper checks
  together for reviewer reproduction. The underlying M5 owner stores remain
  split across the reviewed M5 modules.
- No Spark lane was used; WM-0083 is benchmark/baseline and milestone closeout
  work, and the task packet marks it ineligible.
