## Decisions and deviations

- Roadmap authority remains: M6 = Web / Windows Product Gate, M7 = Early
  Access / public playtest preparation, M8 = 1.0.
- Web remains `demo-only`; Windows remains unsigned
  `ready-for-controlled-external-test`.
- I added automated validation because checklist gate language is high-risk and
  easy to weaken during later edits. The validator checks required snippets
  rather than attempting semantic natural-language review.
- `package.json` quality now includes `pnpm m7:playtest-checklist`, so future
  quality runs protect this protocol.
