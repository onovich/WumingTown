# WM-0129 Owner Blocker Record

## Amendment

Identifier:
`OWNER-AMENDMENT-2026-06-28-POST-M8-UI-PLAYABILITY-REMEDIATION`.

The owner personally ran the current demo and reported the following
release-readiness blockers:

1. Current UI is still too rough for release-level quality.
2. The interface still reads as an internal harness, diagnostics or product
   gate screen rather than a playable game.
3. The visible language experience is mixed Chinese/English.
4. Scene objects, buttons and characters are not reliably mouse-selectable.
5. Scene/camera movement is not drag-operable.
6. Practical player playability is close to zero.
7. M8 closeout must not be interpreted as public release approval.
8. No public release, store, signing, EA, 1.0 launch or final release
   commitment may begin.
9. UI productization, playable interaction, localization, responsive behavior
   and art-direction landing must happen first.

amendment supersedes any interpretation that the current UI is public-release
