# 战斗、治安与守夜

## 定位

战斗是后果制造器，不是主进度。多数异类不能仅靠伤害解决。人类冲突、野兽和少量实体异类使用统一战斗基础。

## 基础链

发现 → 瞄准/接近 → 掩体与视线 → 命中部位 → 防护 → Condition → 疼痛/流血/能力 → 医疗与社会后果。

MVP 使用网格射线与简化弹道，不使用物理引擎作为权威判断。

## 守夜

守夜是职位与工作网络：巡逻路线、灯点检查、名册核验、钟/信号、事故上报、封区。守夜人技能包括观察、纪律、交涉与战斗。

## 警戒层级

日常、注意、封区、全镇警戒。提高警戒会占用劳动力、降低睡眠和贸易，不能长期免费维持。

## 非致命处置

驱离、围困、引导、谈判、撒标记、关闭灯点、隔离居民。UI 优先展示规则匹配处置，不鼓励“全员征召射击”。

## 友军与误认

身份异常会制造误认风险。攻击命令必须显示识别置信度与潜在关系后果；这不是隐藏命中惩罚。
