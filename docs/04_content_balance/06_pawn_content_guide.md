# 居民背景、Trait、关系与证词指南

## 角色构成

```text
出身地区 + 家庭/师承 + 职业经历 + 当前信念
+ 恐惧/执念 + 已有债务 + 具体技能
```

## Trait

Trait 改变决策权重和故事，不应只有 ±10%。例如“守旧”影响对未确认镇志、异类身份和废除镇规的态度；也可能提高旧规执行可靠性。

## 背景

每个背景提供一项能力、一项社会连接和一个潜在冲突。避免“天才”“懒惰”这种扁平标签决定全部人格。

## 证词

证词可受位置、光照、疲劳、伤病、关系、利益和异常影响。文本生成必须区分角色亲眼所见、推断和听说。

## 多样性

角色的地区、年龄、身体状态、职业和家庭结构应多样，但不把身份标签直接绑定道德与能力。残障角色拥有具体能力变化与适应工具，而非只做悲情事件。
