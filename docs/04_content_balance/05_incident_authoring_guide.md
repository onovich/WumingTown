# 事件与危机链创作指南

## 单事件字段

主题、压力、合法条件、前兆、开始窗口、玩家选项、模拟动作、结束条件、后果、恢复提示、冷却、新鲜度标签。

## 危机链

一条好危机链包含：

1. 普通系统弱点，例如灯油不足；
2. 可见前兆；
3. 玩家可采取但有成本的准备；
4. 事件暴露人物关系或制度冲突；
5. 后果进入其他系统；
6. 导演识别压力并给恢复空间；
7. 黎明复盘提供新知识。

## 文本限制

选项文本不能承诺系统没有保证的结果。使用“尝试”“依据当前镇志”并展示风险来源。禁止选完立即弹出一段不可交互小说替代模拟。

## 测试

事件必须在：资源充足、资源短缺、关键角色缺失、目标不可达、存档中断、重复触发和内容缺失场景下验证。
