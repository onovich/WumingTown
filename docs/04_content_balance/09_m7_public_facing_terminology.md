# M7 Public-Facing Terminology Checklist

Status: WM-0101 draft checklist for downstream M7 material. Use together with
`docs/03_world/08_m7_cultural_review_package.md`.

## Required Checks

For every store draft, playtest note, tester checklist, release note, tutorial
copy or screenshot caption:

1. Mark whether the material is internal, controlled external test, public
   draft or final. M7 defaults to draft or controlled external test.
2. Confirm Web is described as `demo-only`.
3. Confirm Windows is described as unsigned controlled external test only.
4. Confirm no line says or implies public release, EA launch, store submission,
   signing, telemetry, account services, paid services, crash upload, final
   privacy/legal/store approval or public save compatibility.
5. Confirm fictional content is not presented as real folklore, history,
   religious practice, medical advice or legal advice.
6. Confirm any real-world inspiration has a `source_note` or is removed.
7. Confirm Chinese and English names use the canonical table below.

## Canonical Terms

| Chinese | English | Notes |
| --- | --- | --- |
| 无明镇 | Wuming Town | Fictional title/place. |
| 镇志 | Chronicle | Public knowledge and evidence system. |
| 灯网 | lamp network | Spatial/civic order system. |
| 镇规 | town ordinance | In-world rule; avoid claiming real law. |
| 旧债 | old debt / obligation | Social/systemic obligation. |
| 异类 | anomalies / anomalous visitors | Avoid defaulting to "monsters". |
| 借影客 | Borrowed Shadow | Fictional anomaly. |
| 第三声 | Third Knock | Fictional anomaly signal. |
| 旧桥客 | Old Bridge Guest | Fictional anomaly/visitor. |
| 黎明复盘 | dawn review | Post-incident evidence review. |

## Blocked Claims

Blocked until owner approval and, when relevant, external review:

- authentic folklore;
- historically accurate ancient China;
- real ritual, real religion or real sacred practice;
- medical, legal or spiritual advice;
- public Web release;
- signed Windows installer;
- public save compatibility;
- final store page;
- Early Access launch;
- paid service or account feature.

## Required Reviewer Questions

Reviewers of WM-0106, WM-0107, WM-0108 and WM-0109 should ask:

- Does the copy distinguish fictional systems from inspiration?
- Does it make any claim that belongs to owner approval?
- Does it turn a protected or sensitive real group/practice into spectacle?
- Does it use "monster" language where "anomaly" or "visitor" is safer?
- Does the English copy add claims that the Chinese copy did not make?
- Does the Chinese copy imply authenticity or a real historical setting?

Any "yes" answer blocks the downstream material until revised or explicitly
owner-approved.
