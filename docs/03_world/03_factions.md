# 派系与社会秩序

## 巡籍司

地方官署的专项机构，负责重建户籍、道路和税契。提供法律承认、正规医疗、工匠与军力；要求公开名册、限制私人异类契约，并可能删除“危险知识”。内部有务实派与正名派。

## 九栈商会

控制多条驿路与客栈网络。提供粮食、灯油、纸张、外地消息和市场；要求道路安全、契约可交易化和商业优先。它可能愿意与夜市合作，但会把旧债商品化。

## 山契诸家

鹤回山不同家族的松散联盟。提供药材、向导、口述历史和本地处置知识；要求尊重祖地、婚姻与互保义务。内部旧仇会让“地方传统”并不统一。

## 夜市诸客

每逢特定无月夜在旧河床出现。提供异常材料、失踪者线索、特殊工人和契约服务；要求条件精确、债务履行和对某些身份的承认。它们不是统一阵营，市场规则只是最低共同秩序。

## 归灯社

无明镇本地的安葬、寻名与遗属互助组织。它可能不是正式宗教。提供葬仪、记忆保存、社区信任；要求死者被妥善记录，不允许以效率为由大量匿名处理。

## 镇议

玩家主要治理载体。由职位、家庭代表和有声望居民构成。不是独立外部派系，但内部倾向会影响制度合法性与执行。
