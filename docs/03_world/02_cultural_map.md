# 文化版图

## 1. 中州与府城文化

重书面名册、法令、标准度量和公开审判。相信统一规则能够消除地方风险。优势是组织、工艺与档案；盲点是把不能分类的关系当作不存在。

游戏表现：官员、医者、书吏、正规器具、稳定贸易与身份审查。

## 2. 灰渡川河路文化

舟民、渡工、渔户和沿河商人重视时令、口令、船上禁忌和互保。身份流动，口头信用比固定土地重要。

游戏表现：水运、鱼盐、消息、失踪者、河界异类与临时盟约。

## 3. 鹤回山诸寨

多个山地家族与小聚落，知识通过师徒、歌谣、绳结和药谱传承。擅长药材、山路与本地契约，但内部血债复杂。

游戏表现：药师、猎人、口述证据、家族信誉、山神土地义务。

## 4. 北原戍户

长期战争形成的屯垦与守备文化。重纪律、同袍和实用防护，对异类态度偏驱逐，但也有独特的战场安魂和无名者登记传统。

游戏表现：守卫、武器、创伤、逃兵、军令与政治压力。

## 5. 南泽社群

湿地与湖沼聚落，以船屋、编织、葬舟和季节迁居闻名。对固定地界观念较弱，认为边界应随水变化。

游戏表现：防潮、草药、纺织、迁徙居民、对镇规的不同理解。

## 6. 夜市诸客

不是单一民族或物种，而是遵守夜市共同规则的异类与人类中介。它们以承诺、故事、影子、名字和稀有材料交易。夜市尊重明确条件，轻视人类单方面解释。

## 设计原则

每个区域都必须同时拥有内部差异、实用知识与局限。不得把“中央”写成纯理性、“地方”写成纯神秘，或把少数文化只作为怪异来源。
