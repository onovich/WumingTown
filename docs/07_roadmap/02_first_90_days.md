# 首 90 天执行建议

以 2 周为一个内部周期，但不对外承诺日期。

## 周期 1：骨架和风险

- Monorepo、CI、严格规则、Worker/Headless 同构
- Pixi Chunk 试验、React Overlay、Electron 安全壳
- Entity/Store、PRNG、World Hash Spike

## 周期 2：地图与实体

- 128×128 网格、Chunk、位置、占用、渲染快照
- 命令协议、选取、开发覆盖层
- 基准框架和内存观察

## 周期 3：路径与工作

- Region、A* 请求、WorkOffer、Top-K
- Job Driver、Reservation 事务
- 失败 ReasonCode

## 周期 4：物流与存档

- 物品、存储、搬运、建造
- Save Container v0、round-trip、重放
- 10 角色长跑

## 周期 5：生活

- 饥饿、休息、食物、基础伤病、能力、医疗
- 工作优先级和日程
- 角色 UI Read Model

## 周期 6：灯火原型

- 灯网、人类秩序场、燃料、守灯 Job
- 黄昏总览第一版
- 夜间路径风险与开发场景

90 天末目标：不是完整游戏，而是可信技术骨架 + 一条生活/物流链 + 灯火空间原型。
