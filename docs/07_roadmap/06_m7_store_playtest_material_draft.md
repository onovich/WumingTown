## Key Claims Allowed In Draft Material

- Fictional single-player town simulation with folk-horror and
  zhiguai-inspired texture.
- Indirect control through planning, work priorities, night watch, records,
  town ordinances and recovery decisions.
- First-season controlled-test focus on onboarding clarity, early resource
  pressure, night-risk readability and structured failure explanations.
- Chronicle, lamp network, town ordinance, obligation and evidence are game
  systems, not claims about real folklore, law, religion or history.
- Windows is ready only for controlled external testing of an unsigned unpacked
  local directory build.
- Web is demo-only in Chrome/Edge product-gate shell form and is not full
  parity.

- Store page, capsule art, trailer, demo page or public recruitment campaign is

- Windows has a signed installer, updater, Steam/store package or public

- Telemetry, account services, paid services, crash upload, hosted feedback or
