# 完成定义

## 所有任务

- 验收标准逐项有证据。
- 类型、Lint、格式和相关测试通过。
- 新行为有测试；Bug 有回归测试。
- 文档、Schema、示例和变更日志同步。
- 无未解决高/中严重度评审发现。
- 报告记录修改、命令、结果、风险和后续。
- 指定线程已收到结构化完成消息。

## 模拟功能额外要求

- 可 Headless 运行。
- 可存档/加载或明确声明不持久状态。
- 可重放且不破坏稳定 Hash。
- 失败产生 ReasonCode。
- 不变量检查覆盖关键关系。
- 性能基准无不可解释回退。

## UI 功能额外要求

- 键盘/鼠标基础可用。
- 缩放、长文本、中英文、无颜色模式检查。
- 不造成全树每 Tick 重渲染。
- Playwright 或截图/行为验证。

## 内容额外要求

- Schema、引用、语义、本地化校验通过。
- 自动场景可触发并结束。
- 文化与公平性检查完成。
- 无专有源码/文本复制。
